# 🏠 hOMER - Real Estate Agency OS

ברוכים הבאים למערכת ניהול סוכנות נדל"ן המתקדמת מבית Omer Digital. 
מערכת זו משלבת ניהול לקוחות (CRM), פייפליין עסקאות דינאמי (Kanban), לוח בקרה הניתן להתאמה אישית, יכולות ייבוא אקסל חכמות, ויצירת קטלוגים משותפים (Webot) עבור הלקוחות.

## 🏗 ארכיטקטורה וטכנולוגיות

המערכת בנויה בארכיטקטורת Serverless חדישה, בגישת Frontend-First:

*   **Frontend (צד לקוח):** React.js + TypeScript, מתארח ב-Vite.
*   **עיצוב ו-UI:** Tailwind CSS, רכיבי Lucide-react לאייקונים. משלב ספריות מתקדמות דוגמת `react-grid-layout` לסידור דאשבורדים ו-`@dnd-kit` לממשק הקנבן של העסקאות.
*   **Backend (צד שרת):** Firebase (Firestore לדאטה-בייס, Firebase Auth לאימות משתמשים, Cloud Storage לתמונות נכסים).
*   **Cloud Functions (פונקציות ענן):** פונקציות Node.js/TypeScript המהוות את הלוגיקה והאבטחה הכבדה של המערכת (Server Authority).

## 🚀 הפיצ'רים המרכזיים במערכת

1.  **Dashboard חכם:** לוח בקרה דינאמי המאפשר למשתמשים לגרור, לעצב, ולשנות את מסכי התצוגה. שמירה אוטומטית לענן.
2.  **עמוד נחיתה (Landing Page):** חלון הראווה הציבורי המציג את תוכניות התמחור, הפיצ'רים והאבטחה של hOMER.
3.  **ניהול מתקדם של נכסים, לידים ועסקאות:** מעקב מלא תוך שימוש במערכת "התאמות חכמות" בין ליד לנכס, חלוקת שלבי מכירה בלוח קנבן אינטראקטיבי.
4.  **יבוא נתונים מ-Excel המוני:** קליטה מהירה של שנות מידע קודמות בממשק אינטואיטיבי המתרגם כותרות באופן אוטומטי לתוך מבנה הנתונים התקני של המערכת (לידים, עסקאות ונכסים).
5.  **WhatsApp Integration & Bulk Messaging:** יכולות לשליחה המונית ללידים, שמירת תבניות למענים חוזרים וסריקת קוד QR ישירות מתוך המערכת.
6.  **סופר אדמין (Super Admin):** מודול נפרד למנהל העל של hOMER למעקב אחרי מדדי כלל המשרדים נרשמו.
7.  **פלטפורמת Webot (קטלוגים שיווקיים):** יצירת קטלוג וירטואלי משותף (Mini-Site) ללקוחות בתוך שניות עם קישור פג-תוקף חכם.

## 🔒 מודל אבטחה (Firestore Rules)

קובץ ה-`firestore.rules` מגן על הנתונים ברמת המסד.
המודל בנוי על עיקרון ה-**Tenant Isolation** – כל סוכן שייך ל-`agencyId` ויכול לגשת **אך ורק** לנתונים של הסוכנות שלו. כתיבת פיצ'רים חדשים מחייבת עמידה במודל זה (העברת `agencyId` אל התהליכים והמסננים).

## 💻 פיתוח ופריסה (Deployment)

**הפעלת סביבת פיתוח (Frontend):**
```bash
cd front
npm install
npm run dev
```

**קימפול פונקציות שרת (Backend):**
```bash
cd back/functions
npm install
npm run build:watch
```

*הערה: פריסה לפרודקשן מבוצעת באמצעות כלי `firebase deploy`.*
