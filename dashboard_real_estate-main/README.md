# 🏠 hOMER - Real Estate Agency OS

ברוכים הבאים למערכת ניהול סוכנות נדל"ן המתקדמת מבית Omer Digital.
מערכת זו משלבת ניהול לקוחות (CRM), פייפליין עסקאות דינאמי (Kanban), בניית קטלוגים דיגיטליים חכמים, חיבור ל-AI (Gemini), דאשבורדים פיננסיים מתקדמים (P&L), ואינטגרציות WhatsApp ו-Stripe. המערכת עוצבה בתצורת Mobile-First, ומספקת חוויית משתמש חלקה גם מהנייד.

## 🏗 ארכיטקטורה וטכנולוגיות

המערכת בנויה בארכיטקטורת Serverless חדישה, בגישת Frontend-First:

*   **Frontend (צד לקוח):** React.js + TypeScript מסביבת Vite.
*   **עיצוב ו-UI:** Tailwind CSS, רכיבי Lucide-react לאייקונים. ספריות متקדמות כמו `react-grid-layout` ו-`@dnd-kit` לממשק הקנבן הייחודי.
*   **Backend (צד שרת):** Firebase צד שרת מלא (Firestore לדאטה-בייס, Firebase Auth, Firebase Storage לתמונות כבדות).
*   **Cloud Functions (פונקציות הליבה v2):** מיקרו-שירותים מבוססי Node.js/TypeScript (למשל: תזמון, גיאו-קוד, Stripe Webhooks, AI, WAHA, וסנכרון יומן Google `googleapis`).

## 🚀 הפיצ'רים המרכזיים במערכת

1.  **Dashboard חכם ומותאם אישית:** מיני-יישומים (ווידג'טים) ניתנים לגרירה וסידור אישי. בנוסף למסך הבית, קיים **דאשבורד רווח והפסד (P&L)** שמציג גרף הכנסות ריאליות (Actual Revenue), ניהול הוצאות חכם (כולל פילוג הוצאות ב-Pie Chart מצורף) וחישובי שולי רווח והפסד בזמן אמת.
2.  **עמוד נחיתה (Landing Page) מובנה:** חלון ראווה ציבורי עם תצוגת פרימיום, המכיל אנימציות גלילה, מחשבון החזר השקעה (ROI), תרחישי תמחור המחוברים הישר ל-Stripe המאפשרים הפעלת סביבה אוטומטית, בנוסף לטופס **יצירת קשר מתקדם** המחובר ישירות למערכת ההפניות.
3.  **אוטומציית רישום וסליקה (Stripe):** רשיונות ומנויים לסוכנויות נדל"ן מנוהלים אוטומטית. תשלום ב-Stripe Checkout מפעיל Webhook שפותח את סביבת הסוכנות, ולוקח את הטלפון לעדכון במסד הנתונים ושולח אימייל בברוכים הבאים.
4.  **ניהול עסקאות והפרדת צדדים (Kanban):** צינור המכירות מאפשר מעקב נוח אחר עסקאות. המערכת תומכת ב**הפרדה מלאה בין נתוני הקונה והמוכר** על גבי כל עסקת נדל"ן לחסכון בזמן ולהימנעות מטעויות כפל ייצוג.
5.  **יבוא חכם מבוסס AI (Gemini):** הזנת טקסט חופשי, קובץ אקסל או קישור למודעה. הבינה המלאכותית מחלצת נתוני נכסים ולידים באופן אוטומטי.
6.  **העוזר החכם (AI Agent):** צ'אט בוט מובנה המבוסס על נתוני הסוכנות בזמן אמת (RAG-lite). ניתן לשאול "מי הליד הכי חם?" או "אילו דירות יקרות יש לנו בנתניה?" וה-AI יענה על סמך המלאי הקיים.
7.  **WhatsApp Integration & AI Firewall:** תקשורת דו-כיוונית מול ווצאפ של הלקוח ישירות מהמערכת (Green API). הבוט מגיע עם **3 שכבות Firewall חכמות לניהול נציג אנושי:**
   * **Auto-Mute בעת מענה אנושי:** אם העובד עונה מרחוק מהטלפון עצמו – הבוט מושתק אוטומטית (כדי לא להיכנס לו לדברים).
   * **Human Handoff:** ה-AI מזהה אם הלקוח מבקש נציג אנושי, מגיב באנושיות שיעבירו אותו לנציג, ומכבה את עצמו.
   * **מתג ידני לשליטה:** ניתן להשתיק או להפעיל ידנית מחדש את הבוט לכל לקוח ישירות מהפרופיל במסך ה-CRM. במקביל, מיון AI יוצר לידים למספרים לא מוכרים ומתייג כוונות.
8.  **Webot (קטלוגים שיווקיים ומיני-סייט):** 
    * אלגוריתם שמתאים אוטומטית נכסים קיימים לדרישות הליד.
    * יצירת מיני-סייט פרטי (Catalog) עם תמונות, מחירים וכפתורי **לייק (❤️)** שומר אל מסד הנתונים את הנכסים המועדפים על הקונה.
    * השליחה נעשית בלחיצה אחת מול הווטסאפ של הקונה.
9.  **Google Calendar Integration:** סנכרון פגישות ופעילויות גיזרום הלידים ישירות מול יומן Google של הסוכן. המערכת קובעת פגישות ומקשרת אותן אסטרטגית לפרופיל הליד/נכס הרלוונטי.
10. **לוח Super Admin:** דאשבורד נפרד לבעלי המערכת, מציג את כלל הסוכנויות (Agencies) הרשומות, נתוני הכנסות, וניהול נתוני המנויים.

## 🔒 מודל אבטחה (Firestore Rules)

*   **Tenant Isolation (הפרדת סביבות):** המודל בנוי כך שכל סוכן יכול בשום אופן לא לגשת לנתונים של סוכנות (Agency) אחרת. 
*   **הרשאות Admin / Agent:** בתוך המשרד, מנהלים יכולים לנהל סוכנים, לייצא נתונים ולשנות הגדרות מערכת, בעוד סוכנים מורשים רק לעבודה השוטפת עם הלידים והעסקאות.
*   **גישה אנונימית לקטלוגים:** המיני-סייטים של הלקוחות (קטלוג) מאובטחים כך שלקוח יכול לקרוא את הקטלוג (עד תפוגת התוקף) ולעדכן בלבד (!) את ה"לייקים" ואת מונה הצפיות ללא צורך בהזדהות.

## 💻 פיתוח ופריסה (Deployment)

**הפעלת סביבת פיתוח (Frontend):**
```bash
cd front
npm install
npm run dev
```

**קימפול ופריסת פונקציות שרת (Backend):**
```bash
cd back/functions
npm install
npm run build
firebase deploy --only functions
```

**עדכון חוקי מסד הנתונים או אחסון דגמים:**
```bash
firebase deploy --only firestore:rules
firebase deploy --only storage:rules
```
